# Jai bindings for tree-sitter

Jai bindings for [tree-sitter](https://github.com/tree-sitter/tree-sitter)

> Note: Only static build

## Usage

```bash
jai generate.jai          # generate bindings
jai generate.jai - clone  # clone and compile tree-sitter
jai generate.jai - clean  # remove cloned tree-sitter repo
```
